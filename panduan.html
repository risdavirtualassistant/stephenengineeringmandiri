1. Upload & Deploy ke GitHub Pages (Hosting Gratis)

Website ini terdiri dari 5 file HTML + 1 file panduan ini. Semua dapat di-host GRATIS di GitHub Pages.

Langkah 1: Buat Akun GitHub

Buka github.com → Klik "Sign up" → Daftarkan email Bapak/Ibu. Nama username akan menjadi bagian URL website.

Langkah 2: Buat Repository Baru

Setelah login → Klik tombol hijau "New" → Isi nama repository, contoh: sem-website → Centang "Public" → Klik "Create repository".

Langkah 3: Upload Semua File HTML

Di halaman repository → Klik "uploading an existing file" → Drag & drop semua file HTML (index.html, tentang.html, jasa-pks.html, katalog-spareparts.html, kontak.html, panduan.html) → Klik "Commit changes".

Langkah 4: Aktifkan GitHub Pages

Klik tab "Settings" → Scroll ke bagian "Pages" → Di "Source" pilih "Deploy from a branch" → Branch: "main" → Folder: "/(root)" → Klik "Save".

Langkah 5: Akses Website

Tunggu 2-5 menit → URL website akan menjadi: https://[username-github].github.io/sem-website/

✅ Website sudah LIVE dan bisa diakses dari Google! File index.html adalah halaman beranda otomatis.

2. Cara Mengedit Konten Website

Ada 2 cara mengedit file HTML — langsung di GitHub (mudah) atau via VS Code (lebih lengkap).

Cara A: Edit Langsung di GitHub (Direkomendasikan untuk perubahan kecil)

  1. Buka repository di github.com
  2. Klik file yang ingin diedit (misal: index.html)
  3. Klik ikon pensil ✏️ di kanan atas file
  4. Edit teks yang diinginkan langsung di browser
  5. Scroll ke bawah → Klik "Commit changes" → Save
  6. Website otomatis terupdate dalam 1-2 menit

Cara B: Edit via VS Code (untuk perubahan besar)

  1. Download file dari GitHub → Edit di VS Code
  2. Temukan teks yang ingin diganti dengan Ctrl+F
  3. Edit → Save → Upload kembali ke GitHub
⚠️ Jangan hapus tag HTML seperti <div>, <section>, atau </>. Hanya edit teks di antara tag tersebut.

Contoh: Mengganti Nomor WhatsApp

Cari teks 6281378293498 di semua file (Ctrl+F), ganti dengan nomor baru. Nomor harus diawali 62 (kode negara Indonesia, tanpa tanda +).

4. Setup Webhook Make.com (Otomatisasi Leads)

Ini adalah sistem yang membuat data dari form website otomatis masuk ke Google Sheets dan mengirim notifikasi ke HP Owner. Gratis!

Langkah 1: Daftar Make.com

Buka make.com → Sign up gratis dengan email → Paket Free sudah cukup untuk 1.000 operasi/bulan.

Langkah 2: Buat Scenario Baru

Klik "Create a new scenario" → Tambahkan modul pertama: cari "Webhooks" → Pilih "Custom webhook" → Klik "Add" → Copy URL Webhook yang dihasilkan.

Langkah 3: Pasang Webhook URL di Semua File HTML

Di setiap file HTML (index.html, jasa-pks.html, katalog-spareparts.html, kontak.html), cari teks YOUR_WEBHOOK_ID dan ganti dengan URL webhook Make.com Anda.

// Cari baris ini di setiap file HTML:
const WEBHOOK = 'https://hook.eu1.make.com/YOUR_WEBHOOK_ID';

// Ganti dengan URL webhook Anda, contoh:
const WEBHOOK = 'https://hook.eu1.make.com/abc123def456ghi789';

Langkah 4: Tambahkan Modul Google Sheets

Di Make.com → Klik "+" → Cari "Google Sheets" → Pilih "Add a Row" → Hubungkan akun Google Anda → Pilih spreadsheet CRM Anda → Mapping: nama→A, hp→B, kebutuhan→C, dst.

Langkah 5: Aktifkan Scenario

Klik toggle "ON" di kiri bawah Make.com → Scenario sekarang aktif dan akan berjalan setiap ada form yang diisi.

5. Google Sheets CRM — Formula Otomatis

Buat Google Sheets baru sebagai CRM. Gunakan formula di bawah untuk membantu analisis leads.

Struktur Kolom yang Disarankan:

A: Waktu · B: Nama · C: Perusahaan · D: HP · E: Jenis Kebutuhan · F: Urgensi · G: Detail · H: Halaman · I: Status · J: Prioritas (auto)

Formula: Flag URGENT + Governor Woodward (Kolom J)

=IF(AND(ISNUMBER(SEARCH("URGENT",F2)),ISNUMBER(SEARCH("Governor",E2))),"🔴 HIGH PRIORITY",IF(ISNUMBER(SEARCH("URGENT",F2)),"🟡 URGENT",IF(ISNUMBER(SEARCH("Governor",E2)),"🔵 WOODWARD","🟢 Normal")))

Formula: Hitung Total Leads per Bulan

=COUNTIFS(A:A,">="&DATE(YEAR(TODAY()),MONTH(TODAY()),1),A:A,"<"&DATE(YEAR(TODAY()),MONTH(TODAY())+1,1))

Formula: Hitung Leads Governor Woodward

=COUNTIF(E:E,"*Governor*")
✅ Tempel formula di baris pertama kolom J, lalu drag ke bawah untuk semua baris. Status otomatis ter-update saat data baru masuk.

6. Koneksi WhatsApp Notifikasi ke Owner

Saat ini, form website otomatis membuka WhatsApp dengan pesan pre-filled untuk dikirim ke Owner. Untuk notifikasi otomatis tanpa klik, gunakan cara berikut di Make.com:

Opsi A: Notifikasi via Email (Gratis)

Di Make.com → Tambahkan modul "Email" → Kirim ke email owner setiap ada lead masuk. Subject: "[LEAD BARU] {{nama}} - {{kebutuhan}}"

Opsi B: Notifikasi via Telegram Bot (Gratis)

Buat Telegram Bot via @BotFather → Tambahkan modul "Telegram Bot" di Make.com → Kirim pesan ke nomor HP owner setiap ada form terisi. Ini adalah cara paling cepat.

Opsi C: WhatsApp via Twilio/WATI (Berbayar)

Untuk notifikasi WhatsApp otomatis 100%, gunakan layanan WATI.io atau Twilio WhatsApp API. Hubungkan ke Make.com dengan API key.

💡 Rekomendasi: Mulai dengan Telegram Bot (gratis, instan) sambil sistem berjalan. Upgrade ke WhatsApp API saat volume leads sudah tinggi.
